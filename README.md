Bank Loan Modelling Analysis
Project Overview

This project analyzes customer banking data to identify factors that influence personal loan acceptance. The analysis includes exploratory data analysis (EDA), statistical testing, data visualization, and predictive modeling techniques.

Objectives
Analyze customer demographics and financial characteristics
Identify factors affecting personal loan acceptance
Perform correlation and statistical analysis
Visualize customer behavior patterns
Build predictive models for loan acceptance

Key Insights
Income showed a strong relationship with loan acceptance.
Customers with higher education levels were more likely to accept loans.
Credit card spending and CD accounts influenced loan conversion rates.
Machine learning models helped predict potential loan customers.
